Immediate help

E-mail sharing service not sending attachments?

This issue is related to either of the three possibilities:

  1. There is poor Internet connection to the FOCUSA server. Check the Internet connection to the server.
  2. The e-mail account must have exhausted it's subscribed resources. This means that nobody can use the e-mail service until the account gets re-activated.
  3. The e-mail SMTP configurations have been set incorrectly. Inform the network administrator to change the configurations.

Student has forgotten their password?

Do not worry! Administrators can make use of the updateUser() command in the dashboard. This command only requires 3 inputs: the username, the new account password, and the account's new permissions(or access controls, which can be set the same as previous).

Nobody can comment on posts?

This could mean that commenting is disabled by the network administrator. Inform the network administrator to change the configurations.

Introduction

FOCUSA is a Content Management System designed similar to a social networking site, like reddit with a twist. FOCUSA manual pages is an attempt made to create an interactive set of manuals that can help a new user to make their life easier. Also, since you are the end user of this software, I have given you a method of communicating with me related to any new feature, new bug, or help in a topic not stated in these manual pages.

How to use these manual pages

This webpage is designed to be simple and easy to use. You just have to click on a link on the contents pane, and the document related to the link clicked will open here. Another important functionality of this webpage, is that it allows administrators to type in administrative commands. More information can be found at the dashboard manual pages.
